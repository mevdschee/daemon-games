daemon-games
============

![screenshot](http://www.leaseweblabs.com/wp-content/uploads/2014/11/snaked.png)

Multi-player command line games for GNU/Linux

### Requirements

- C compiler
- GNU/Linux
- netcat (nc)

### Installing and running

```
git clone https://github.com/mevdschee/daemon-games
cd daemon-games
make
./snaked 9000
./client.sh 0 9000
```
